更新文章
hexo clean //删除之前的静态文件
hexo g //重新生成新的
hexo s //开启服务
    

提交代码
git init 初始化git
git add . 添加文件
git commit -m '备注'
git pull origin master

如果出现合并操作
键盘输入‘i’
键盘输入‘esc’
键盘输入‘:wq’
键盘输入‘enter’，
提交代码成功

服务部署到github
代码更新到git服务之后在git bush中执行hexo clean清除缓存，然后执行hexo g -d生成hexo静态网页并部署
